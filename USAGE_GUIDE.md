# 软考刷题插件使用指南

## 1. 试题管理功能

### 1.1 综合试题管理
1. 在IDE中按 `Ctrl+Alt+E` 直接打开综合试题管理界面
2. 在界面中可以：
   - **添加单个试题**：手动添加新试题（统一表单界面）
   - **批量导入试题**：从JSON/XML文件导入试题
   - **查看模板示例**：查看标准试题模板格式
   - **管理知识点章节**：管理试题的章节分类
   - **查看所有试题**：浏览所有已添加的试题
   - **编辑试题**：修改试题信息（复用添加界面）
   - **删除试题**：移除不需要的试题
   - **查看详情**：只读模式查看试题完整信息（复用表单界面）

### 1.2 试题导入功能

### 1.1 JSON模板格式
JSON文件必须是一个数组，包含一个或多个试题对象。每个试题对象的结构如下：

```json
{
  "id": "唯一标识符",
  "title": "题目内容",
  "options": {
    "A": "选项A内容",
    "B": "选项B内容",
    "C": "选项C内容",
    "D": "选项D内容"
  },
  "correctAnswers": ["A"],  // 正确答案，可以是单个或多个选项
  "explanation": "题目解析",
  "level": "简单|中等|困难",  // 难度等级
  "category": "分类名称",
  "year": 2023,  // 年份
  "examType": "系统分析师|系统架构设计师|网络规划设计师|信息系统项目管理师|系统规划与管理师|系统集成项目管理工程师|网络工程师|信息系统管理工程师|软件评测师|数据库系统工程师|多媒体应用设计师|软件设计师|信息系统监理师|电子商务设计师|信息安全工程师|嵌入式系统设计师|软件过程能力评估师|计算机辅助设计师|计算机硬件工程师|信息技术支持工程师|程序员|网络管理员|信息处理技术员|信息系统运行管理员|多媒体应用制作技术员|电子商务技术员|网页制作员"  // 考试类型
}
```

### 1.2 如何导入试题
1. 准备好符合上述格式的JSON文件
2. 在IDE中按 `Ctrl+Alt+Q` 打开试题管理功能
3. 选择"查看模板示例"查看标准模板格式
4. 选择"批量导入试题"
5. 选择"JSON"格式
6. 选择您的JSON文件
7. 点击"导入"完成操作

### 1.3 模板示例
参考 `src/main/resources/templates/questions_template.json` 文件获取完整的使用示例。

## 2. 试题导出功能

### 2.1 导出数据类型
- **练习记录**: 包括练习次数、答题总数、正确数、正确率等
- **错题本**: 包括错题ID、错误次数、最后错误时间、掌握状态等
- **学习统计**: 包括总体统计数据、分类统计、获得成就等

### 2.2 如何导出数据
1. 在IDE中按 `Ctrl+Alt+E` 打开导出功能
2. 选择导出路径
3. 选择导出类型（练习记录、错题本或学习统计）
4. 点击"导出"完成操作

## 3. 手动添加试题功能

### 3.1 如何添加单个试题
1. 在IDE中按 `Ctrl+Alt+Q` 打开试题管理功能
2. 选择"添加单个试题"
3. 按以下步骤填写表单：
   - **题目标题**: 输入题目内容
   - **分类**: 输入题目分类（如Java基础、Spring框架等），默认为"未分类"
   - **难度**: 从下拉菜单选择难度（简单、中等、困难），默认为"中等"
   - **选项**: 输入选项标识（如A、B、C、D）和选项内容
     - 系统默认提供A-D选项
     - 至少需要2个有效选项
   - **选择正确答案**: 在下方复选框中勾选正确选项
   - **解析**: 输入题目解析说明（可选）

### 3.2 填写说明
- **题目标题**: 必填，输入题目内容
- **分类**: 必填，默认为"未分类"，建议按知识领域分类
- **难度**: 必选，默认为"中等"
- **选项**: 
  - 左侧输入选项标识（如A、B、C、D）
  - 右侧输入选项内容
  - 至少需要2个选项，最多支持4个
- **正确答案**: 必选0，勾选所有正确选项（支持单选和多选）
- **解析**: 可选0，输入解题思路或知识点说明

## 4. 快捷键列表

- `Ctrl+Alt+P`: 打开练习窗口
- `Ctrl+Alt+W`: 打开错题本窗口
- `Ctrl+Alt+Q`: 打开试题管理功能
- `Ctrl+Alt+E`: 打开数据导出功能

## 5. 功能模块说明

### 5.1 练习模式
- **每日一练**: 随机生成10道题目
- **专项练习**: 按分类练习
- **模拟考试**: 生成50道题目进行模拟考试
- **随机练习**: 随机抽取题目进行练习

### 5.2 学习统计
- 总练习次数
- 答题总数和正确数
- 正确率计算
- 连续学习天数
- 分类统计
- 获得成就

### 5.3 错题本
- 自动记录错题
- 错题错误次数统计
- 掌握状态追踪
- 错题重做功能
