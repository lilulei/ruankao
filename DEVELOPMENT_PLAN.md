# IDEA软考刷题插件开发计划

## 需求完成情况对比表

### 2.1 试题管理模块
- [x] 默认试题库（内置软考历年真题）
- [x] 按考试科目分类
- [x] 支持试题标签体系（知识点、难度等级、年份）*通过QuestionNoteService实现*
- [x] 自定义试题（手动添加单个试题）*通过AddQuestionDialog实现*
- [x] 批量导入功能（支持JSON、XML格式）*通过ImportQuestionsDialog和QuestionImportUtil实现*
- [x] 试题编辑与删除功能 *通过QuestionService实现*
- [x] 试题预览和验证机制 *通过JSON模板实现*
- [x] JSON模板支持 *通过questions_template.json实现*

- [x] 答题界面

- [x] 选项单选/多选支持

- [x] 上一题/下一题导航
